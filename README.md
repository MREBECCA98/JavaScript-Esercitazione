JavaScript 

Questo progetto contiene una serie di esercizi di JavaScript svolti per il corso di Epicode.
Gli esercizi coprono concetti di base e avanzati di JS, tra cui:
-Variabili e operazioni
--Oggetti e array
-Funzioni
-Manipolazione di stringhe e numeri
-Operazioni con date
-Cicli e condizioni
-Interazioni base con il DOM

Tecnologie utilizzate:
-JavaScript (ES6+)
-HTML5 (solo per includere il JS)

Note:
La maggior parte degli esercizi è orientata alla pratica della logica in JavaScript, non alla visualizzazione su pagina.
Questo progetto è stato realizzato come esercitazione didattica.
